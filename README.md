# Elshenawy Restaurant Consulting

موقع عربي احترافي لخدمات تأسيس وتشغيل وتطوير وتقييم المطاعم في المملكة العربية السعودية ودول الخليج، بقيادة محمد الشناوي.

## الملفات والصفحات

- الصفحة الرئيسية: `index.html`
- من نحن: `about.html`
- الخدمات: `services.html`
- طلب خدمة: `request.html`
- تواصل معنا: `contact.html`
- المقالات: `articles.html`
- صفحة 404: `404.html`
- صفحات الخدمات: داخل مجلد `services/`
- ملفات SEO: `robots.txt` و `sitemap.xml`

## إضافة خدمة جديدة

أضف بيانات الخدمة في ملف البناء `work/build_consulting_site.mjs` داخل قائمة `services` ثم أعد تشغيل البناء.

## إضافة سعر لاحقًا

يمكن استبدال عبارة "اطلب عرض سعر" أو "يتم تحديد التكلفة بعد معرفة تفاصيل المشروع ونطاق العمل" داخل صفحة الخدمة أو إضافة قسم باقات مستقل.

## تغيير الاسم مستقبلًا

غيّر قيمة `brand` و `arabicBrand` في ملف البناء. سيظهر الاسم الجديد في الهيدر والفوتر وبيانات SEO وSchema.

## إضافة الشعار الحقيقي

استبدل الشعار النصي `ERC` بصورة شعار داخل الهيدر والفوتر، ثم أضف ملف الشعار إلى المشروع وحدّث Schema.
