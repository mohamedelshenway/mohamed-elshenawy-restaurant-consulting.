# الشناوي لاستشارات المطاعم

موقع عربي احترافي لخدمات استشارات المطاعم في المملكة العربية السعودية، بقيادة محمد الشناوي.

## التقنية المستخدمة

الموقع ثابت Static Website مكون من HTML وCSS وJavaScript بدون مكتبات خارجية ثقيلة.

## تعديل بيانات التواصل

عدّل القيم التالية داخل ملف البناء:

- `phone`
- `phoneDisplay`
- `whatsapp`

ثم أعد تشغيل ملف البناء.

## تغيير الشعار

الشعار موجود داخل:

- `assets/logo-horizontal.svg`
- `assets/logo-icon.svg`

يمكن استبدالهما بشعار حقيقي بنفس الأسماء.

## تعديل الخدمات

الخدمات موجودة في مصفوفة `services` داخل ملف:

`work/build_elshenawy_clean_site.mjs`

## إضافة الأسعار لاحقًا

يمكن إضافة حقول سعر داخل كل خدمة أو إنشاء قسم باقات مستقل. حاليًا لا توجد أسعار معلنة.

## إضافة مقال

استخدم صفحة `articles/template.html` كقالب، ثم أضف رابط المقال في `articles.html` و `sitemap.xml`.
