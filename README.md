# مِعيار المطاعم

موقع عربي لخدمات استشارات وتقييم وتشغيل وتطوير المطاعم داخل المملكة العربية السعودية.

## الهوية

- الاسم: مِعيار المطاعم
- الوصف: للاستشارات والتشغيل والتطوير
- الشعار التسويقي: نقيّم بدقة ونطوّر بخطة ونشغّل بنظام
- المؤسس: محمد الشناوي
